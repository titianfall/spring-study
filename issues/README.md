# Issues

Codex에 물어본 내용과 강의에서 헷갈린 내용을 이슈 단위로 저장하는 공간입니다.

## 작성 규칙

- 파일명은 `NNNN-topic.md` 형식을 사용합니다.
- GitHub issue와 연결하려면 파일 안에 `learning-issue-id` 주석을 유지합니다.
- 최종 학습 기록은 Markdown 파일 자체를 기준으로 관리합니다.
- 여러 챕터를 한 번에 정리할 때는 챕터별로 커밋을 나누고, 하나의 PR로 묶습니다.

예:

```text
issues/0001-project-environment.md
```
