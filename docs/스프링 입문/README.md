## 작성 규칙

- 파일명은 강의 챕터 번호와 제목을 그대로 쓴 한글 파일명(`NN. 챕터 제목.md`)을 사용합니다.
- 학습 **전**에 강의자료 PDF를 기반으로 내용을 미리 정리합니다.
- 학습하며 실습 코드는 챕터 단위로 커밋하고, 브랜치에서 커밋 내용을 근거로 md를 보충한 뒤 main에 merge합니다.
- 최종 학습 기록은 Markdown 파일 자체를 기준으로 관리합니다.
